### Week 1 Achievements:

* Understand what AWS is and master the basic service groups: Compute, Storage, Networking, and Database.

* Successfully created and securely configured an AWS Free Tier account.

* Familiarized with the AWS Management Console and learned how to search, access, and use services through the web interface.

* Installed and configured AWS CLI on the computer, including: Access Key, Secret Key, Default Region, and Output format (JSON/Table).

* Used AWS CLI to perform basic operations such as: Check account information & environment configuration, Access the list of active Regions, View the status of EC2 services, Create and manage secure Key Pairs, and Check information about services running in the background.

* Gained the ability to flexibly connect between the web interface and CLI to manage AWS resources in parallel.
